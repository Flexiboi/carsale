Accasion script to sell vehicles player to player with ingame shop creator.
</br>
It will display NUI ontop of the vehicle displaying the price.
</br>
Player will need to drive to the ped and press **E** to sell the vehicle if it is their own.
</br>
When all spots taken player will not be able to sell.
</br>
When spot is free it will take free spot to sell / display vehicle.
</br>
Players can check the vehicles and view info like description of seller and upgrades.
</br>
Setup mail when vehicle is bought.
</br>

**Commands**
</br>
- **addcarlocation** -> Add a new sale location
- **addsalespot** -> Add a new sale spot based on closest location
- **removesalespot** -> Removes closest sale spot
- **removelocation** -> Removes closest sale location

</br>
</br>

**Setup**
</br>

- Use qbox or make it work for any other
- Need ox lib
- Need sql script
